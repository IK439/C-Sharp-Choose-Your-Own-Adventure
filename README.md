# Choose Your Own Adventure (C#)

A simple **text-based horror adventure game** written in C#.  
The player explores a drifting space station and makes choices that affect the outcome of the story.

## Description

This program is a console-based **Choose Your Own Adventure** game that uses:
- User input
- Conditional logic (`if`, `switch`)
- Loops for input validation
- Basic methods

The story is set in a **sci-fi horror** environment where the player investigates strange sounds aboard an abandoned space station.

## How to Run

1. Open the project in **Visual Studio** or another C# IDE.
2. Build and run the program.
3. Follow the on-screen prompts and type your choices when asked.

## Features

- Interactive story choices
- Input validation (prevents invalid responses)
- Multiple branching paths
- Horror-themed narrative
- Simple and readable C# structure

## Technologies Used

- C#
- .NET Console Application
